# Agent reports

Agents write authored reports here.

Raw commands, logs, screenshots, and machine-generated artifacts belong in `../results/`.

Neither folder proves completion. Record verified status in the active checklist and active blockers in `../Failures.md`.

## Report buckets

- `admin/` - admin work, checks, and admin UI passes.
- `ds/` - design sheets and the DS index.
- `issues/` - issue packs and the issues index.
- `ui/` - UI passes and the UI flawless report.
- `reviews/` - code, config, features, i18n, verification, and audit reviews.
- `ops/` - fix, dispatch, and execution reports.
