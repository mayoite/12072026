# World-class UI bar (owner-enforced)

**This is not a slogan.** Owner paid real money and time. Scaffolding = failure.

## Reference quality (aim here)

Think **Linear settings + Stripe Dashboard + Figma properties** — not “admin CRUD demo.”

| Principle | Concrete |
|-----------|----------|
| Hierarchy | One clear title, one primary action, quiet secondary |
| Density | Comfortable, not sparse CAD void; not cramped |
| Surface | Solid panel surfaces; never graph paper under forms |
| Type | Clear label → control; muted meta; no engineer essays in chrome |
| Preview | Product-quality plate: framed, centered, aspect-respecting SVG |
| Color | Brand tokens only; calm ecru/paper + primary CTA |
| Motion | Subtle or none; respect reduced-motion |
| Honesty | No fake controls, no dual editors, no dual primary buttons |
| Empty | Helpful one line, not dump |

## Parametric factory must feel like

**“Configure a desk and publish it for the customer.”**  
Not “debug Maker multipath IR.”

## Fail if any remain

- Grid under form fields  
- “form + Maker” / Engine:… slogans in chrome  
- Eye/pencil/list legend theater  
- Two equal Publish buttons  
- Identity editable in two places  
- Tiny grey desk in a huge empty void  
- Native-looking cheap inputs with no focus craft  
- Hex hardcodes in chrome CSS  

## Pass only when

A non-engineer would say: *this looks like a real product I could use to sell desks.*
