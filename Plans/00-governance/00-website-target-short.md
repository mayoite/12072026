# One&Only Website Target - Short

Date: 2026-07-07

Status: short target definition.

Source truth: `https://oando.co.in`.

## Target

Build the best possible website for One&Only.

One&Only is an office furniture and workspace solutions business.

The website must sell trust, products, planning, projects, and service.

It is not a carpet site.

It is not a flooring site.

It is not only a planner.

## What The Website Must Do

1. Explain One&Only clearly.
2. Show products and solutions.
3. Prove project capability.
4. Let users plan workspaces.
5. Convert visitors into briefs and quotes.
6. Let admins manage business content and assets.
7. Track leads and project work.
8. Ship with evidence.

## Main Modules

| Module | Purpose |
| --- | --- |
| Public site | Brand, products, solutions, projects, proof, and quote flow. |
| Planner | Workspace planning, furniture placement, save/export, and quote intent. |
| Admin | Products, projects, SVG blocks, planner content, CRM, assets, and users. |
| SVG system | Clean, safe, versioned layout and product blocks. |
| CRM | Leads, contacts, customers, quotes, follow-ups, and project status. |
| Auth | Guest, member, partner, sales, admin, and super-admin access. |
| Database | Safe business data with migration discipline. |
| CDN/assets | Images, SVGs, downloads, and project media. |
| Deployment | Build, lint, tests, console, and release evidence. |

## Quality Bar

The site should feel premium, calm, exact, and business-grade.

Reference quality:

1. Steelcase for workspace storytelling.
2. MillerKnoll for premium design authority.
3. Herman Miller for product trust.
4. Haworth for catalog and design resources.
5. IKEA planners for simple planning entry.
6. RoomSketcher and Planner 5D for planning UX.
7. Shopify admin for backend clarity.

## Ideal End State

The ideal One&Only website:

1. Looks premium.
2. Explains the business fast.
3. Shows products clearly.
4. Proves real delivery.
5. Lets users plan workspaces.
6. Converts visitors into quote-ready briefs.
7. Gives admins real control.
8. Keeps planner and SVG assets clean.
9. Supports CRM follow-up.
10. Ships with proof, not guesses.
