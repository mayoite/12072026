## Rule

Do not run multiple phases as one hidden batch.

Each phase needs:

1. Entry condition.
2. Allowed scope.
3. Checks.
4. Exit evidence.
5. Stop conditions.
6. Refusals and deferrals.
7. Artifact path, if a gate ran.
