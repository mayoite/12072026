This folder is the planner program record. It is not a claim that the product is done.

1. [Active execution plan](./execution/00-ACTIVE-EXECUTION-PLAN.md) — current work order and proof required for each phase.
2. [Start and operating truth](./01-START-HERE.md) — product outcome, unlock, approach, and hard limits.

## Folder map

| Folder | Purpose | Use it for |
|---|---|---|
| `execution/` | Current, revised work plan | Choose the next phase. Do real work. |
| `phases/` | Original detailed P01–P10 phase cards | Implementation detail and phase acceptance criteria. |
| `checkpoints/` | Gate definitions | Deciding whether a phase is actually complete. |
| `checklists/` | Cross-phase controls | Required checks and agent rules. |
| `history/` | Superseded reviews and deferred ideas | Context only. Never execute it as a current plan. |
