## How to Work Together on a Project in Planner 5D

1. ### Open your project


Go to your dashboard and open the project you want to share.

2. ### Open the Share menu


In the project toolbar, click the Share icon and select Share Project.

3. ### Invite collaborators


In the sharing window, add people by email or copy the link to share it directly. For each person you invite by email, choose their permission level: view or edit.

4. ### Manage access anytime


After sharing, you can review and update each collaborator's permissions at any time, upgrading, downgrading, or removing access as needed.

5. ### Collaborator gets notified


Invited users receive an email notification with a direct link to join the project. One click and they're in.

## Frequently Asked Questions

### What is shared collaboration mode in Planner 5D?

Shared collaboration mode in Planner 5D lets multiple people work on the same home design project simultaneously in real time. Whether you're designing with a partner, getting feedback from a client, or collaborating with a colleague, everyone sees changes as they happen. Each participant has their own cursor and presence indicator, so you always know who\\'s working on what.

### Can multiple people edit a home design at the same time?

Yes. With Planner 5D's multiplayer feature, several people can edit the same project at the same time from different devices. Changes made by any collaborator appear instantly for everyone in the session, no need to refresh, export, or share files manually.

### How do I share my project with a collaborator?

You can invite collaborators directly from your project by sharing a link or sending an email invitation. Once they join, they're immediately part of the session and can start contributing based on the permissions you\\'ve assigned.

### Can I control what collaborators can do — view, comment, or edit?

Yes. Planner 5D's multiplayer mode supports role-based permissions. You decide whether each collaborator can only view the project, leave comments, or make edits. This makes it easy to share work-in-progress with clients for feedback while keeping full editing control for yourself.

### Do my collaborators need a Planner5D account to join?

Collaborators can join a shared project via a link. Depending on their role, account requirements may vary. Check the sharing settings in your project for the most up-to-date access options.

### Is shared editing available on all plans?

Yes, shared editing is available on all Planner 5D plans, including free. You can invite anyone to your project at no extra cost. Guests can view the design without an account, but if a collaborator wants to actively edit the project, they'll need to create a free Planner5D account first.

### What happens if two people edit the same object at the same time?

Planner 5D handles simultaneous edits automatically to keep your project consistent. If two collaborators interact with the same object at the same moment, the system resolves the conflict so no work is lost. You'll always see the current state of the design in real time.
