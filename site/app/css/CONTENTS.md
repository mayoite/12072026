# Shared CSS (FOCSS)

## What is here

- `index.css` — entry (`theme` → `utilities` → `components`)
- `core/theme.css` — `@theme` tokens
- `core/utilities/*.css` — typography, animations, layout, colors, schemes, buttons, badges, nav, misc
- `core/components/*.css` — shared component styles (extract on 3rd use)
- `core/locked/{site,admin,planner}/` — surface modules
