# Core CSS

Three base files at `core/` root, then surface modules under `core/locked/`.

```text
app/css/
  index.css
  core/
    theme.css       # @theme tokens (single source)
    utilities.css   # typography, animations, layout/color/scheme helpers
    components.css  # shared component patterns
    locked/
      site/
      admin/
      planner/
```
