## 5. Gate definitions (binding — copy from phase, expand meaning)

### 5.1 W1 — Draw structure

| Field | Binding definition |
|-------|--------------------|
| **Must prove** | Walls metric **increases** after user/agent draw · door/opening **increases objects** after place |
| **Must not pass on** | Seed walls ≥1 or ≥4 alone · toast/copy alone · absolute object count without post-opening delta |
| **Route** | `/planner/open3d` preferred · guest fallback |
| **Proof** | Playwright + PNGs `02-walls-drawn.png` · `03-door-opening.png` (+ route-ready earlier) |
| **Tools** | Tool labels **Wall** then **Opening** (not shortcut D) |

**Why deltas (false-green killer):** Guest **Start from Scratch** may already have **≥4 walls**. Absolute `walls ≥ 1` is a lie. Capture `wallsBefore` → draw four segments (or enough geometry to raise count) → assert `wallsAfter > wallsBefore`.

### 6.3 W1 interaction script (semantic, not code dump)

1. **Baseline:** `wallsBefore = getWallCount` · `objectsBefore = getObjectCount`  
2. **Select Wall tool** via `selectPlannerTool(page, "Wall")` matching label pattern  
3. **Draw rectangle-ish four segments** with `dragOnCanvas` relative coords, e.g.:
   - (0.25, 0.25) → (0.75, 0.25)  
   - (0.75, 0.25) → (0.75, 0.75)  
   - (0.75, 0.75) → (0.25, 0.75)

5. Screenshot **`02-walls-drawn.png`**  
6. Note `wallsAfterDraw` · `objectsAfterWalls`  
7. **Select Opening tool** (label **Opening**, not keyboard D)  
8. **placeOpeningOnCanvas** near a wall mid-span, e.g. (0.45, 0.25) → (0.55, 0.25)  
   - Fallback: single `tapOnCanvas` on wall if placeOpening fails once  
9. **Poll** objects **> objectsAfterWalls**  
10. Assert walls still **≥ wallsAfterDraw** (opening must not destroy wall geometry)  
11. Screenshot **`03-door-opening.png`**
