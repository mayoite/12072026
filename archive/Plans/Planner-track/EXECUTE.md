# Planner execute

**Order:** P01 → P16. One active Planner card.

1. Confirm dependencies.
2. Write a failing behavior test.
3. Implement on the live Fabric host.
4. Run targeted units and typecheck.
5. Run the card browser journey when UI changes.
6. Check the shared UI bar.
7. Update the card, checkpoints, and board.

Evidence is a dump. Live commands decide PASS.

Stop for a product fork, owner waiver, destructive cleanup, or commit/push.
