## 4. How to build each phase (recipe)

For **each** P01–P10, one agent or human does:

### Step A — Verdict
Open `plans1/P0X/CODE-REVIEW-REPORT.md` **and** `plans2/P0X/CODE-REVIEW-REPORT.md`.  
Write one line:

```
GATE: FAIL | APPROVE residual | RE-PROVE only
REASON: <evidence missing / code residual / freeze>
```

### Step B — Residual contract (max 12 bullets)
From:
- plans1 EXECUTABLE-PLAN task group for that phase

```markdown
# P0X residual contract
**Evidence:** results/planner/world-standard-wave/<folder>/
**Posture:** code residual | re-prove | harden | pack
**Do:**
- [ ] …
**Do not:**
- [ ] rewrite engine / mesh / invert / …
**Stop if:** …
**Done when:** map-min files + command logs on this HEAD
```

### Step C — Drop the dump
Do **not** paste full IMPL TDD novels into the one plan.
